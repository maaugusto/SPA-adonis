# AdonisJS SPA

A Monolith SPA scaffolding with AdonisJS, Inertia JS/React and Tailwind.

## Use

```shell
  git clone git@github.com:maaugusto/SPA-adonis.git
  cd SPA-adonis
  npm install
  cp .env.example .env
  node ace generate:key
  node ace serve --watch
```

Feel free to use it in your projects.
Cheers!
